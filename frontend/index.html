Worker URL (your Cloudflare Worker)
Deploy the worker from GitHub repo, then paste the URL here. This is what handles AI, search, Gmail, and Obsidian.
Puter (free Claude/GPT — no key needed)
Used as fallback when Worker is unavailable. No key needed — sign in with Puter once.
Gmail Setup
One-time setup: In Google Cloud Console → Create OAuth 2.0 credentials → Add these to your Worker env: GMAIL_CLIENT_ID, GMAIL_CLIENT_SECRET. Then visit /auth/gmail on your worker to connect your Gmail account.
Obsidian / GitHub Sync
One-time setup: Create a GitHub repo for your Obsidian vault (or use your existing one). Create a GitHub Personal Access Token with repo scope. Add GITHUB_TOKEN and GITHUB_REPO (e.g. copeland/my-vault) to Worker env.
Search Provider
Add SERPER_KEY to Worker env for web search. Free at serper.dev (2,500 searches/month).
